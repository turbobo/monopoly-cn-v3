# 大富翁中国行 - 游戏规则（在线版）

## 游戏概述

大富翁中国行是一款精简版大富翁桌游，以中国城市为主题，支持三种游戏模式：

| 模式 | 说明 |
|------|------|
| 👥 本地多人 | 同一设备2-4人轮流操作 |
| 🤖 挑战AI | 1v1/1v2/1v3 AI对战 |
| 🌐 在线对战 | GoEasy 实时通信，最多4人在线对战 |

玩家通过掷骰子移动、购买地皮、收取租金、使用道具卡来积累财富，最终成为最富有的玩家。

---

## 游戏模式详解

### 👥 本地多人模式

**玩法**：同一设备2-4人轮流操作，适合朋友聚会、家庭娱乐。

**特点**：
- 所有玩家在同一台设备上轮流操作
- 每回合自动切换到下一位玩家
- 支持暂停/继续功能
- 无网络依赖，离线可玩

**设置**：
- 玩家数量：2-4人
- 初始资金：¥800-¥10000（8档预设 + 自定义）
- 游戏时长：♾️无限 / 20 / 30 / 50 / 100 回合

---

### 🤖 挑战AI模式

**玩法**：1名人类玩家 vs 1-3名AI对手，单人挑战模式。

**AI对手**：
| 难度 | AI名称 | 性格 | 特点 |
|------|--------|------|------|
| 🌱 简单 | 老守 | 保守型 | 只买便宜地皮，资金充裕才出手 |
| ⚖️ 普通 | 阿平 | 平衡型 | 看性价比，保留安全资金 |
| 🔥 困难 | 小火 | 激进型 | 只要买得起就买，积极扩张 |

**特点**：
- AI 自动决策：掷骰、移动、购买、道具卡使用
- 支持暂停/继续功能
- 无网络依赖，离线可玩

**设置**：
- AI对手数量：1-3个
- 游戏难度：🌱简单 / ⚖️普通 / 🔥困难
- 初始资金：¥800-¥10000（8档预设 + 自定义）
- 游戏时长：♾️无限 / 20 / 30 / 50 / 100 回合

### 游戏难度详解（AI模式）

难度影响 AI 的**买地决策阈值**、**拍卖出价上限**、**交易接受底线**、**主动变卖阈值**、**监狱出狱策略**等所有决策维度。

| 难度 | 难度系数 | AI性格 | 买地阈值 | 拍卖上限 | 交易底线 | 变卖阈值 | 出狱策略 |
|------|---------|--------|---------|---------|---------|---------|---------|
| 🌱 简单 | ×1.5 | 保守型 | 资金 ≥ 价格×1.8×1.5 | 价格×0.6 | 价格×1.2 | ¥150 | 第3回合强制 |
| ⚖️ 普通 | ×1.0 | 平衡型 | 资金 ≥ 价格×1.2×1.0 | 价格×0.8 | 价格×1.3 | ¥100 | 第2回合主动 |
| 🔥 困难 | ×0.7 | 激进型 | 资金 ≥ 价格×0.8×0.7 | 价格×0.95 | 价格×1.5 | ¥50 | 第1回合主动 |

**难度系数说明**：
- **简单（×1.5）**：AI 更保守，需要更多资金才购买，适合新手玩家
- **普通（×1.0）**：AI 正常决策，平衡风险与收益
- **困难（×0.7）**：AI 更激进，资金紧张时也积极购买，适合挑战极限

难度**不影响**租金计算、机会卡、道具卡等其他游戏规则。

### 🌐 在线对战模式

**玩法**：通过 GoEasy 实时通信，最多4人跨设备在线对战。

**特点**：
- 跨设备联机：手机、平板、电脑均可加入
- 实时同步：所有玩家看到相同的游戏状态
- 房主权威模式：房主执行所有游戏逻辑，保证状态一致
- 消息压缩：广播时只发送最近20条消息，减少流量
- 断线重连：自动心跳检测，断线后自动重连

**连接方式**：
1. **创建房间（房主）**：输入名字 → 点击「🏠 创建房间」→ 获得6位房间号
2. **加入房间（玩家）**：输入名字 → 输入房间号 → 点击「加入」
3. **开始游戏**：房主看到所有玩家就绪后，点击「开始游戏」

**房号机制**：
- 房号为自动生成的6位随机码（不含易混淆字符 I、O、0、1）
- 可点击复制按钮分享给朋友
- 每个房间最多4人

**游戏同步**：
- **房主权威模式**：房主执行所有游戏逻辑（掷骰、移动、购买判定、道具卡等），将完整游戏状态广播给所有玩家
- **客户端操作**：当轮到某位玩家时，其掷骰/购买/道具卡操作会发送给房主，由房主执行后广播新状态
- **等待提示**：非当前回合的玩家看到「⏳ 等待 XXX 操作...」

**断线处理**：
- 玩家加入/离开：播放声音提示（加入=上升三连音，离开=下降三连音）
- 玩家断线：房主自动将其标记破产并广播更新
- 房主断线：其他玩家收到「与房主的连接已断开」提示，返回主菜单

**设置**：
- 玩家数量：2-4人（由加入房间的人数决定）
- 初始资金：¥800-¥10000（8档预设 + 自定义）
- 游戏时长：♾️无限 / 20 / 30 / 50 / 100 回合
- **无难度设置**（所有玩家均为人类）

## 在线对战模式

在线模式使用 GoEasy PubSub 实现实时通信。所有游戏逻辑在浏览器中运行，可直接部署到 EdgeOne Pages 免费版等纯静态托管平台。

## 回合流程

1. **掷骰子**：投掷两颗骰子（各1-6点）
2. **移动**：按点数总和前进相应格数（棋子沿棋盘路径逐格移动，带抛物线弧形+旋转动画）
3. **经过起点**：获得 ¥200 奖励（随回合递增，每5回合+¥50，上限¥400）
4. **执行操作**：根据到达格子的类型执行对应操作

## 购买规则

到达无主地皮时：
- **人类玩家**：弹出购买确认框，可选择购买或放弃
- **AI玩家**：根据性格+难度自动决策（仅AI模式）
- **在线模式**：远程玩家的购买决策由房主等待10秒，超时自动跳过

### AI性格类型（仅AI模式）

| 性格 | 决策逻辑 |
|------|---------|
| 🔥 激进型 | 只要买得起就买（资金 ≥ 价格×0.8×难度系数） |
| 🛡️ 保守型 | 只买便宜且有余钱（资金 ≥ 价格×1.8×难度系数 且价格 ≤ ¥200） |
| 🤖 平衡型 | 看性价比，保留安全资金（资金 ≥ 价格×1.2×难度系数） |

**难度系数**：简单=1.5（更保守）、普通=1.0（正常）、困难=0.7（更激进）

### AI 进阶策略（仅AI模式）

- **同色组进度加成**：AI 已拥有同色组 ≥50% 时购买意愿大幅提升，已有 ≥1 块时也有加成
- **回合数加成**：游戏进入中后期（>10回合）AI 更激进，>20回合后更激进
- **主动变卖资产**：资金低于阈值时（激进¥50/平衡¥100/保守¥150），优先卖出不在同色组内的最便宜地皮
- **主动发起交易**：AI 会主动寻找能补齐同色组的地皮，向其他玩家出价购买
- **拍卖决策**：AI 参与拍卖时，激进型最高出价×0.95、保守型×0.6、平衡型×0.8
- **交易决策（卖方）**：AI 出售地皮时，激进型最低接受×1.5、保守型×1.2、平衡型×1.3
- **监狱出狱策略**：激进型第1回合主动支付保释金、平衡型第2回合支付、保守型等到第3回合强制出狱

## 监狱规则

1. **入狱方式**：
   - 到达"入狱"格
   - 机会卡"进监狱"

2. **出狱方式**：
   - 掷出双数（两个骰子点数相同）→ 出狱并按该点数移动
   - 等待3回合 → 缴纳 ¥50 保释金自动出狱
   - **AI 主动出狱**：激进型第1回合主动支付保释金、平衡型第2回合支付、保守型等到第3回合强制出狱

3. **监狱中**：
   - 每回合掷骰子尝试出狱
   - 未出双数则跳过回合

## 破产规则

当玩家资金变为负数时：

1. **自动卖地**：从最便宜的地皮开始，6折出售
2. **持续卖地**：直到资金 ≥ 0 或地皮卖光
3. **破产判定**：卖光地皮仍为负数 → 破产，退出游戏
4. **零资产判定**：现金为0且无地皮 → 同样判定破产

## 胜利条件

满足以下任一条件即获胜：

1. **淘汰胜利**：其他所有玩家破产（任何模式下均有效）
2. **资产胜利**：达到回合上限时，总资产最高者获胜（仅设置了回合上限时）

**总资产计算** = 现金 + 地皮原价总和

**游戏时长选项**：

| ♾️ 无限（默认） | 纯淘汰制，只剩1人存活才结束 |
| 20回合 | 快速局，到期按资产排名 |
| 30回合 | 标准局 |
| 50回合 | 长局 |
| 100回合 | 史诗局 |

## 道具卡系统

### 获取方式

每5个游戏回合（第5、10、15...回合），所有未破产玩家各获得1张随机道具卡。

### 使用条件

- 只能在自己的回合、掷骰子之前（phase=roll）使用
- 在线模式下 Guest 的道具卡操作会发送给 Host 执行
- 不能在特殊格子（起点、监狱、免费停车、入狱）放置路障

### 道具卡类型

| 道具卡 | emoji | 效果 | 需要选择 |
|--------|-------|------|---------|
| 🎯 遥控骰子 | 🎯 | 指定本次掷骰点数（2-12） | 选择点数 |
| 🚧 路障卡 | 🚧 | 在指定格子放置路障，其他玩家踩中后本次移动无效 | 选择格子 |
| 🔄 交换卡 | 🔄 | 与任意未破产玩家互换棋盘位置 | 选择玩家 |
| 🛡️ 免费卡 | 🛡️ | 激活后下次需付租金时免除费用 | 直接激活 |
| 📈 涨价卡 | 📈 | 自己的一块地皮租金翻倍，持续3个游戏回合 | 选择自己的地皮 |

### 路障规则

- 路障为一次性效果，踩中后自动移除
- 不会触发自己放置的路障
- 踩中路障的玩家本回合直接结束，不触发格子效果
- 不允许在特殊格子（起点、监狱、免费停车、入狱）放置路障

### 涨价卡规则

- 只能对自己拥有的地产使用
- 效果持续3个游戏回合后自动失效
- 租金翻倍与回合加成、同色组加成叠加计算

### 地图标记

- 🚧 路障：格子背景变为橙红色调，右上角显示🚧角标
- 📈 涨价：格子背景变为金黄色调，左上角显示📈角标

## 技术信息

- **技术栈**：Next.js 14 + TypeScript + Tailwind CSS + GoEasy
- **在线通信**：GoEasy PubSub 实时消息（房主权威模式）
- **渲染**：HTML5 Canvas 实时渲染（深蓝主题+织纹+粒子系统+道具卡地图标记）
- **音效**：Web Audio API 合成音效（含玩家加入/离开提示音）
- **动画**：逐格移动动画（squash&stretch + 360°旋转 + 25px抛物线）
- **部署**：EdgeOne Pages 静态导出（output: 'export', outputDirectory: "out"）
- **测试**：Vitest 单元测试（game-engine + online-utils）
