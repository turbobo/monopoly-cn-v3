# 大富翁中国行 - 游戏规则（在线版）

## 游戏概述

大富翁中国行是一款精简版大富翁桌游，以中国城市为主题，支持三种游戏模式：

| 模式 | 说明 |
|------|------|
| 👥 本地多人 | 同一设备2-4人轮流操作 |
| 🤖 挑战AI | 1v1/1v2/1v3 AI对战 |
| 🌐 在线对战 | P2P直连，最多4人实时对战（无需服务器） |

玩家通过掷骰子移动、购买地皮、收取租金来积累财富，最终成为最富有的玩家。

---

## 在线对战模式

### 基本原理

在线模式使用 PeerJS（WebRTC）实现 P2P 直连，无需后端服务器。所有游戏逻辑在浏览器中运行，可直接部署到 EdgeOne Pages 免费版等纯静态托管平台。

### 连接方式

1. **创建房间（房主）**：输入名字 → 点击「🏠 创建房间」→ 获得房间号
2. **加入房间（玩家）**：输入名字 → 输入房间号 → 点击「加入」
3. **开始游戏**：房主看到所有玩家就绪后，点击「开始游戏」

### 房号机制

- 房号即房主的 PeerJS ID（格式：`monopoly-{时间戳}-{随机串}`）
- 可点击复制按钮分享给朋友
- 每个房间最多4人

### 游戏同步

- **房主权威模式**：房主执行所有游戏逻辑（掷骰、移动、购买判定等），将完整游戏状态广播给所有玩家
- **客户端操作**：当轮到某位玩家时，其掷骰/购买操作会发送给房主，由房主执行后广播新状态
- **等待提示**：非当前回合的玩家看到「⏳ 等待 XXX 操作...」

### 断线处理

- 玩家断线：房主自动将其从房间移除并广播更新
- 房主断线：其他玩家收到「与房主的连接已断开」提示，返回主菜单

## 基础设置

| 项目 | 说明 |

### 游戏难度（AI模式）

| 难度 | AI买地策略 | 适合人群 |
|------|-----------|---------|
| 🌱 简单 | AI更保守（资金需≥价格×1.5倍才买） | 新手玩家 |
| ⚖️ 普通 | AI正常决策（资金需≥价格×1.0倍） | 中等体验 |
| 🔥 困难 | AI更激进（资金需≥价格×0.7倍就买） | 挑战极限 |

难度仅影响AI的买地决策阈值，不影响租金和其他规则。在线模式无难度设置。

**右侧（7-13格）**：监狱探访 → 重庆 → 西安 → 国家电网 → 长沙 → 机会 → 杭州

**顶部（14-20格）**：免费停车 → 成都 → 广州 → 房产税 → 南京 → 机场 → 深圳

**左侧（21-27格）**：入狱 → 苏州 → 机会 → 天津 → 上海 → 北京 → 中国移动

## 回合流程

1. **掷骰子**：投掷两颗骰子（各1-6点）
2. **移动**：按点数总和前进相应格数（棋子沿棋盘路径逐格移动，带抛物线弧形+旋转动画）
3. **经过起点**：获得 ¥200 奖励（随回合递增，每5回合+¥50，上限¥400）
4. **执行操作**：根据到达格子的类型执行对应操作

### 租金随回合加成

| 回合 | 租金倍率 |
|------|---------|
| 1-9回合 | ×1.0（原价） |
| 10-19回合 | ×1.5 |
| 20-30回合 | ×2.0 |

### 起点奖金随回合递增

| 回合 | 经过起点奖金 |
|------|------------|
| 1-5回合 | ¥200 |
| 6-10回合 | ¥250 |
| 11-15回合 | ¥300 |
| 16-20回合 | ¥350 |
| 21回合起 | ¥400（上限） |

## 购买规则

到达无主地皮时：
- **人类玩家**：弹出购买确认框，可选择购买或放弃
- **AI玩家**：根据性格+难度自动决策（仅AI模式）
- **在线模式**：远程玩家的购买决策由房主等待10秒，超时自动跳过

### AI性格类型（仅AI模式）

| 性格 | 决策逻辑 |
|------|---------|
| 🔥 激进型 | 只要买得起就买（资金 ≥ 价格×0.8×难度系数） |
| 🛡️ 保守型 | 只买便宜且有余钱（资金 ≥ 价格×1.8×难度系数 且价格 ≤ ¥200） |
| 🤖 平衡型 | 看性价比，保留安全资金（资金 ≥ 价格×1.2×难度系数） |

**难度系数**：简单=1.5（更保守）、普通=1.0（正常）、困难=0.7（更激进）

## 监狱规则

1. **入狱方式**：
   - 到达"入狱"格
   - 机会卡"进监狱"

2. **出狱方式**：
   - 掷出双数（两个骰子点数相同）→ 出狱并按该点数移动
   - 等待3回合 → 缴纳 ¥50 保释金自动出狱

3. **监狱中**：
   - 每回合掷骰子尝试出狱
   - 未出双数则跳过回合

---

## 破产规则

当玩家资金变为负数时：

1. **自动卖地**：从最便宜的地皮开始，6折出售
2. **持续卖地**：直到资金 ≥ 0 或地皮卖光
3. **破产判定**：卖光地皮仍为负数 → 破产，退出游戏
4. **零资产判定**：现金为0且无地皮 → 同样判定破产

---

## 胜利条件

满足以下任一条件即获胜：

1. **淘汰胜利**：其他所有玩家破产
2. **资产胜利**：30回合结束，总资产最高者获胜

**总资产计算** = 现金 + 地皮原价总和

## 技术信息

- **技术栈**：Next.js 14 + TypeScript + Tailwind CSS + PeerJS
- **在线通信**：WebRTC P2P（PeerJS信令服务器 + Google STUN）
- **渲染**：HTML5 Canvas 实时渲染（深蓝主题+织纹+粒子系统）
- **音效**：Web Audio API 合成音效
- **动画**：逐格移动动画（squash&stretch + 360°旋转 + 25px抛物线）
- **部署**：EdgeOne Pages 静态导出（output: 'export', outputDirectory: "out"）
